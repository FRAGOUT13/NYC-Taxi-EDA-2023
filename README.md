# NYC Yellow Taxi EDA 2023
EDA_Optimising_NYC_Taxis_Arnab_Kanu

## Project Overview
Exploratory Data Analysis of NYC Yellow Taxi trip data for 2023.
Analysed patterns in taxi operations to optimise service efficiency,
maximise revenue and enhance passenger experience.

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- GeoPandas
- Jupyter Notebook

## Key Findings
- JFK Airport is the busiest pickup zone (~96,015 trips)
- Peak hours are between 5PM and 9PM
- Vendor 2 (VeriFone) charges slightly more per mile than Vendor 1
- Average tip percentage is 24.63% for credit card trips
- Q2 and Q4 contribute the most revenue at 26.7% each

## Dataset
The dataset is not included due to large file sizes.
Download the 2023 Yellow Taxi trip records from:
https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

Place the 12 monthly parquet files in a folder called 'data'
and the shapefile parts in a folder called 'taxi_zones'.

## How to Run
1. Clone this repository
2. Set up the data folder as described above
3. Open the notebook in Jupyter
4. Run all cells from top to bottom
